[NAVBAR Главная :: index.html, SQL :: sql_page.html] # О Haskell, корохотенько ![https://img.devrant.com/devrant/rant/r_424296_voADE.jpg](intro) Исполнить код на haskell можно с помощью [https://downloads.haskell.org/~platform/2014.2.0.0/](платформы). Там есть - компилятор (ghc) - профилятор (hp2ps) - repl (ghci) - генератор документации (haddock) - менеджер БД об установленных пакетах (ghc-pkg) - сборщик статистики о том сколько раз использовалось (hpc) - инструмент для спайки кода на haskell и C (hsc2hs) - и запускаторы компилятора и интерпретатора (runghc, runhaskell) В repl приглашение `Prelude> ` значит, что уже сразу подключен модуль `Prelude`. Можно поменять строку приглашения `:set prompt "жопа> "`, или повводить выражения `33+3*3`, `"ABC"++"DEFG"`. Как текстовый редактор можно поставить [http://mirror.tochlab.net/pub/gnu/emacs/windows/emacs-27](emacs) с расширением spacemacs. ### spacemacs на шindows После установки emacs, делаем бекап папки `C:/Users//AppData/Roaming/.emacs.d` ![https://simpletutorials.com/c/spacemacs/mqt72al0/static/4.png](переименовали в .emacs.d.old) И [https://git-scm.com/download/win](git)-ом клонируем всё необходимое ``` git clone https://github.com/syl20bnr/spacemacs .emacs.d ``` ![https://simpletutorials.com/c/spacemacs/mqt72al0/static/5.png](клонируем spacemacs git-ом) У emacs на windows проблемы с https, поэтому для обновлений его нужно запустить так: Win + R ``` runemacs --insecure ``` Затем, нужно выбрать раскладку (vim или emacs) и полную, либо облегчённую версию spacemacs. Spacemacs скачает и поставит всё что нужно.
