[NAVBAR Главная :: index.html, SQL :: sql_page.html] # О Haskell, корохотенько ![https://img.devrant.com/devrant/rant/r_424296_voADE.jpg](intro) Исполнить код на haskell можно с помощью [https://downloads.haskell.org/~platform/2014.2.0.0/](платформы). Там есть - компилятор (ghc) - профилятор (hp2ps) - repl (ghci) - генератор документации (haddock) - менеджер БД об установленных пакетах (ghc-pkg) - сборщик статистики о том сколько раз использовалось (hpc) - инструмент для спайки кода на haskell и C (hsc2hs) - и запускаторы компилятора и интерпретатора (runghc, runhaskell) В repl можно поменять строку приглашения `:set prompt "жопа> "`
