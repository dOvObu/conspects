[NAVBAR Главная :: index.html, SQL :: sql_page.html, Haskell :: hs_page.html, Анализ_данных :: data_analysis.html] # О линейной алгебре, корохотенько ![https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcT31W8uTJNIq5WUg9YWDrIgHuo-Z3bv5YIQFPaX8_BbWZZHocRpJAxJebBVjgAKun2_QD4&usqp=CAU](lineman) ### Линейное пространство \(V\) над \(CC\) def Для \(vec u in V\) определена
операция сложения \(+ :: VxxV -> V\), которая = коммутативна, ассоциативна, для неё \(EE vec 0 in V: vec 0 + vec u = vec u\)
и \(AA vec u in V: EE vec u^-1 in V : vec u + vec u^-1 = vec 0\) def Для \(vec u in V\) так же определена
операция умножения \(* :: CCxxV -> V\), которая = ассоциативна, дистрибутивна относительно сложения
и для неё \(EE 1 : 1 * vec u = vec u\) К примеру, линейное пространство может состоять из векторов, или полиномов вида \(f(x) = a_0 + a_1*x+...+a_n*x^n\) ### Группа def Группа \(G\) это множество \(x\) на котором введена
операция \(@ :: x xx x -> x\), которая = ассоциативна,
\(EE e in X: AA a in X: e @ a = a\)
и \(AA a in X: EE a^-1 in X: a @ a^-1 = e\) К примеру, матрицы вида \(A = ((a_1, 0),(0, a_2))\) не могут быть группой,
потому что если \(a_2 = 0\), то обратной матрицы \(A^-1\), такой чтоб \(A*A^-1 = E\), уже нигде не найдётся. А вот матрицы такого же вида, но с дополнительным требованием \(a_1 != 0\) и \(a_2 != 0\), уже группа. ### Линейная зависимость def вектора \(vec v_1, vec v_2, ... , vec v_k\) линейно зависимы, если = \(EE a_i != 0: a_1 vec v_1 + ... + a_k vec v_k = 0\) К примеру, если нашёлся такой коэффициент \(a_i\), то вектор при котором он стоит, можно выразить разделив левую и правую часть уравнения на \(a_i\), ведь \(a_i != 0\). \(] a_1 != 0\) \[a_1 vec v_1 + a_2 vec v_2 + ... + a_k vec v_k = 0\] \[a_1 vec v_1 = - a_2 vec v_2 - ... - a_k vec v_k\] \[vec v_1 = (- a_2 vec v_2 - ... - a_k vec v_k)/a_1\] А вот геометрический пример. Три вектора на плоскости всегда линейно зависимы. def \(vec v_1, vec v_2, ... , vec v_k\) линейно независимы, если = \(a_1 vec v_1 + ... + a_k vec v_k = 0\) только при \(AA i in [1..k]: a_i = 0\) К примеру, два неколлинеарных вектора на плоскости -- линейно независимы, поэтому их можно использовать как базисные вектора: [https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTe46nIwJuMAZJTAVJL54FCrv93nlzdlnjHPA&usqp=CAU]()
