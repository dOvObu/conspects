это разбиение `QQ` на два непустых множества `A и A' : (A cap A' = O/), (A cup A' = QQ)` и что самое главное `forall x in A, forall x' in A': x < x'`.


разбиение `alpha` записывается как `A|A'`

Примеры:

  1. `A = {x in Q : x <= 1}, A'={x in Q : x > 1}`
  2. `A = {x in Q : x < 1}, A'={x in Q : x >= 1}`
  3. `A = {x in Q : x < 0} cup {x in Q : 0 < x, x^2 < 2}, A'={x in Q : 0 < x, 2 < x^2}`

Другого варианта нет
границы `r_1=max A, r_2=min A'` не подходят, т.к. не равны: `r_1 < r_2`,
между ними `r_0 in (r_1, r_2)` не подходят, т.к. не входят в куски разбиения `r_0 notin A и r_0 notin A'`


`RR` это просто сечения второго и третьего типов.


для `alpha, beta in RR`, сечения `alpha = A|A'` и `beta = B|B'` равны, если `A=B`, либо, что то же самое, `A'=B'`
`alpha` больше `beta`, если `A sup B <=> A' sub B'` (у кого нижний класс меньше, тот раньше начался)
`alpha` меньше `beta`, если `A sub B <=> A' sup B'`


Если `alpha ne beta` то, либо `alpha < beta`, либо `beta < alpha`.

От обратного

пусть неверно, что `beta < alpha`, то значит `EE r_1 in QQ: r_1 in B, r_1 in A'`
а если неверно, что `alpha < beta`, то значит `EE r_2 in QQ: r_2 in A, r_2 in B'`
Получается, что `r_1 in B, r_2 in B'`,то есть `r_1 < r_2`,
а `r_2 in A, r_1 in A'`, то есть `r_2 < r_1`.
