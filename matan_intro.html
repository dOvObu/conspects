Конспекты


это разбиение `QQ` на два непустых множества `A и A' : (A cap A' = O/), (A cup A' = QQ)` и что самое главное `forall x in A, forall x' in A': x < x'`.


#define `A|A'` :: разбиение `alpha`

Примеры:

  1. `A = {x in Q : x <= 1}, A'={x in Q : x > 1}` :: в `A` есть верхний элемент, а в `A'` нет нижнего
  2. `A = {x in Q : x < 1}, A'={x in Q : x >= 1}` :: в `A` нет верхнего, в `A'` есть нижний
  3. `A = {x in Q : x < 0} cup {x in Q : 0 < x, x^2 < 2}, A'={x in Q : x > 0, x^2 > 2}` :: нет, нет | т.к. корень из двух -- число не рациональное

Другого варианта НЕТ
границы `r_1=max A, r_2=min A'` не подходят, т.к. не равны `r_1 < r_2`,
между ними `r_0 in (r_1, r_2)` не подходят, т.к. не входят в куски разбиений `r_0 notin A и r_0 notin A'`
