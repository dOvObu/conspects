Конспекты

`A={a_1, a_2, ..., a_n}`


это упорядоченный набор из `n` разных `a_i in A`
Всего их `P_n = n! = 1 * 2 * ... * n`

это как перестановка, но длиной может быть меньше чем `n`
Всего их `A_n^k = (n!)/((n-k)!) = (1 * 2 * ... * (n - k) * ... * n)/(1 * 2 * ... * (n - k)) = (n - k + 1) * (n - k + 2) * ... * n`

это размещение, в котором могут быть повторения
Всего их `stackrel(~)(A_n^k) = n^k = n * n * ... * n`


это размещение, в котором не важен порядок
Всего их `((n),(k)) = (n!)/(k! (n-k)!) = (1*2*...*(n-k)*...*n)/((1*2*...*k)*(1*2*...*(n-k))) = ((n - k + 1)*...*n)/(1*2*...*k) = (A^k_n)/(P_k)`


это размещение с повторениями, в котором не важен порядок
Всего их `stackrel(~)(((k),(n))) = ((n + k - 1)(k)) = ((n + k - 1)!)/(k! (n - 1)!)`
