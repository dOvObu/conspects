это упорядоченный набор из `n` разных символов
Всего их `P_n = n! = 1 * 2 * ... * n`
Похоже на круговорот воды.

это упорядоченный набор из `k <= n` разных символов
Всего их `A_n^k = (n!)/((n-k)!) = (1 * 2 * ... * (n - k) * ... * n)/(1 * 2 * ... * (n - k)) = (n - k + 1) * (n - k + 2) * ... * n`
Похоже на воронку в чашке с чаем.

Всего их `stackrel(~)(A_n^k) = n^k = n * n * ... * n`
Похоже на слова в европейских языках

это неупорядоженный набор из `k <= n` возможно повторяющихся символов
Всего их `((n),(k)) = (n!)/(k! (n-k)!) = (1*2*...*(n-k)*...*n)/((1*2*...*k)*(1*2*...*(n-k))) = ((n - k + 1)*...*n)/(1*2*...*k) = (A^k_n)/(P_k)`
Похоже на виды рыбы, которая попадает в сети.
